# Vue Template

Created to quickly start a new Vue app with my standard defaults

### Dependencies

[yarn](https://yarnpkg.com/en/docs/install), dependency manager

### Project setup

Change remote url:

```
git remote set-url origin https://github.com/USERNAME/REPOSITORY.git
```


### Development

Start developing
```
yarn run dev
```

Lint
```
yarn run lint
```

Build
```
yarn run build
```
